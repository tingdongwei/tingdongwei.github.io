博客搭建记录

发表于 2021-03-04 分类于 技术

欢迎来到阿五的黑历史世界！它是由Hexo搭建的 。所有源码都放在了上GitHub 。关于这个博客 ， 很多基本格式还需要熟悉 ， 我选择从头构建 ， 组件会逐渐添加 。下面是这次搭建博客的具体步骤

搭建hexo静态博客

下面以win为例，mac可以参考CodeSheep的视频 。

下载基本工具

首先需要下载nodejs,选择下载左侧的LTS（长期支持版） 。 安装到任意想要的位置，推荐默认c盘 。

win+r输入cmd打开窗口 ， 输入查看是否安装好，以及node版本 。

安装git

接着下载 git ,可以选择 Windows , macOS , Linux/Unix 几种版本，安装到想要的任意位置。其中默认桌面右键可以打开Git Bash 。

设置淘宝镜像

同理，设置好后，可以通过cnpm -v查看 (不想看版本请忽略)

正式安装基础框架

设置好后，可以通过hexo -v查看版本

在自己想要的位置新建一个文件夹，并在窗口进入其中。这里我是在c盘建立的testblog文件夹，并输入hexo init

其中这里比较容易出现问题 ， 并通常需要较长时间的等待 。

常见问题的解决方案有

  1. 在建好的文件夹下右键打开Git Bash ， 输入hexo init （ 强烈推荐 ）
  2. cmd中输入cnpm install , 接着hexo s
  3. 如果hexo d 出现报错 Deployer not found: git ， cmd中输入hexo npm install hexo-deployer-git –save ， 手动添加git
  4. cmd中输入cnpm install git

运行博客

到这里博客就基本搭建完成了 ！ 让我们打开运行一下 ，输入hexo s测试运行 。

打开浏览器输入4000端口进入本地预览网页，输入localhost:4000 , 就可以看见建好的博客了，它会自动创建一篇文章 “ Hello World ” 。

新建文章

通过hexo -n可以新建文章 ，例如这里我建立了一篇名为测试文章的博文 。

也可以通过在根目录文件夹（\source_posts），右键新建文本文档，通过改后缀txt为md文件的方式，新建文章。

编写文章

以我建立的名为testblog根目录为例 ， md文件储存位置为（C:\Users\Administrator\testblog\source_posts）

下载VSCode编辑器 ， 并安装到任意位置 ， 打开VSCode , 把需要编写的md文件拖入VSCode内 ， 在其中编辑好后ctrl+s保存即可 。

只要是可以编辑markdown格式的软件或在线程序都可以

上传文章

以名为testblog根目录为例，（C:\Users\Administrator\testblog\）输入hexo cl清理一下 ， 后输入hexo g生成 ， 再hexo s重新启动即可。

推送远端

通过把博客部署到 github上 ， 实现免费远端公开访问.

首先我们需要注册一个github账号 ， 并登录 ， 点击右上角加号点击New repository 新建一个仓库 。

在新建仓库时 ， 名字一定要和昵称保持一致 ， 仓库名应为 ：’ 你的github昵称.github.io ‘ ， 一定要符合这个要求 ！

添加描述后 ， 进入新建的空仓库 ， 复制HTTPS的地址 ， 在根目录下输入cnpm install hexo-deployer-git –save安装git的插件 。

打开根目录中的_config.yml文件 ， 在最下面补充完整。并ctrl+s保存 。

deploy:
type: ‘git’
repo: https://github.com/你的昵称/你的昵称.github.io.git
branch: master

这时就可以输入hexo d 运行了。并在弹出的git窗口输入邮箱和密码。

也打开git bash，并在命令窗口中输入git config –global user.name “xxx” 。 再输入git config –global user.email “xxx@xxx”设置好邮箱昵称和密码 。注意：git需要设置用户名和邮箱地址 ， 否则hexo d会报错 。

常见问题：

  1. 报错也可以在最后添加一行message: Blog解决 。
  2. Spawn failed可以重新添加git。

在浏览器输入网址https://github.com/你的昵称/你的昵称.github.io.git并稍等刷新 ， 即完成部署博客 。

更换主题

我们可以在hexo的主题官网找到很多好看的主题 。 这里列举一些常见的好看主题 。 以yelee主题为例 ， 在根目录下输入git clone https://github.com/MOxFIVE/hexo-theme-yelee.git themes/yelee就可以在themes文件夹下找到名为yelee的主题文件夹 。打开根目录下的_config.yml文件 ， 找到theme：landscape修改为目标主题。

并重新启动服务
